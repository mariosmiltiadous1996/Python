*Kivy*
Intall steps:


1)Download the latest python version and install to your pc

2)Download latest library version of pygame and install

3)Open cmd and write the follow commands:
	i) pip install --upgrade pip wheel setuptools
       ii) pip install docutils pygments pypiwin32 kivy.deps.sdl2 kivy.deps.glew kivy.deps.gstreamer --extra-index-url https://kivy.org/downloads/packages/simple/
      iii) pip install kivy

4)Call kivy from idle python with "import kivy.app"

5)Ready to run the project code using "F5"
